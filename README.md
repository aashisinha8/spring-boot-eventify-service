# Eventify - Smart Event Management System

Spring Boot backend service for managing events and users with role-based access.

## Tech Stack
- Java 17
- Spring Boot
- Spring Data MongoDB
- MongoDB
- REST APIs

## Features
- Create and manage users (Admin, Organizer, Participant)
- Create and manage events
- Store data in MongoDB
- REST API testing via Postman

## APIs

### Health Check
GET /health

### Create User
POST /users

### Create Event
POST /events
